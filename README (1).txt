# Laptop Buying Data Analysis

## Overview
This project analyzes laptop pricing data to identify trends and patterns based on various features such as brand, RAM, and screen size. The analysis is performed using Python with data visualization libraries.

## Features
- **Data Loading**: Reads laptop data from a JSON file.
- **Price Distribution**: Displays a histogram to understand the pricing spread.
- **Brand Analysis**: Shows the average laptop price per company.
- **RAM vs. Price Correlation**: Visualizes the relationship between RAM size and laptop price.
- **Screen Size vs. Price**: Explores how screen size affects pricing.

## Requirements
Install the necessary Python libraries before running the notebook:
```bash
pip install pandas matplotlib seaborn
```

## Usage
1. Place your dataset (JSON format) in the appropriate location.
2. Open and run the Jupyter Notebook.
3. Explore the visualizations and insights generated.
